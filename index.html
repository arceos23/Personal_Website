About this website

At the start of building this website, I decided to not use any libraries or frameworks with the intention to get more experience with the fundamentals of web development and have the utmost freedom and flexibility to create my website. After completing the website, I have gained a much greater appreciation for libraries and frameworks as the speed of development is much faster when using them, especially to someone with less experience building a user interface from scratch. Nonetheless, I learned a lot from this experience and exercised my hacking skills to get things to work. New things I worked on that I did not have experience with before include:

  1. Using semantic HTML elements such as headers, footers, and sections which improve code readability, website accessibility, and SEO performance
  2. Toggling CSS styles by using the DOM to implement the responsive nav bar that turns into a toggle menu at small enough screen sizes
  3. Manipulating the DOM to implement the selected project carousel which changes the selected portfolio either by the arrow buttons or slide indicator

Currently, this website is hosted on [GitHub Pages]. Next, I want to complete the Cloud Resume Challenge to gain hands-on experience with cloud technologies and DevOps. I have inherently completed some of this challenge since I have already created this personal website and obtained the AWS Certified Cloud Practitioner certification. Completing this challenge would give me practical exposure with:

  1. DNS
  2. Infrastructure as Code
  3. CI/CD
